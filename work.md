## 정리(클린) 버전: LH 50억 미만 적격검사 기본값 초안
### 경영점수(방법1: 요소 합산)
- 부채비율점수: (자체 부채비율 ÷ 업종평균 부채비율)
  - < 50%: 7.0점, < 75%: 6.2점, < 100%: 5.4점, < 125%: 4.6점, ≥ 125%: 3.8점
- 유동비율점수: (자체 유동비율 ÷ 업종평균 유동비율)
  - ≥ 150%: 7.0점, ≥ 120%: 6.2점, ≥ 100%: 5.4점, ≥ 70%: 4.6점, < 70%: 3.8점
- 영업기간점수(가산): 3년 이상 1.0점, 1년 이상 0.9점, 1년 미만 0.8점
- 총점 = 부채(최대7) + 유동(최대7) + 영업기간(최대1) = 15점 만점

### 경영점수(방법2: 신용평가 환산)
- 환산식: 기준점수 × (15/35)
- 예: AAA~BBB0 → 35.0 → 15.000, BB- → 34.5 → 14.786, B계열 → 34.0 → 14.571, CCC~D → 30.0 → 12.857
- 유효기간 등 공고 기준 충족자료만 사용
 대답: 기능 화면에서 사용자가 공고일을 등록하도록 할거야

### 시공경험 점수
- 실적계수 × 13, 실적계수 = (최근5년 실적 ÷ 기초금액)
- 제안: 상한 13점 캡(확인 필요) 
 대답: 상한 13점 맞아

### 협정(공동도급)
- 경영점수: 구성원별 경영점수 산정 후 지분율 가중합산
- 시공점수: (Σ(5년실적_i × 지분_i) ÷ 기초금액) × 13

### 기본값 적용 범위(초안)
- 발주처: 한국토지주택공사(LH)
- 금액구간: 0 ≤ 금액 < 5,000,000,000원

## 제안
- 산업평균: 현재 코드의 INDUSTRY_AVERAGES 사용하되, 발주처별 커스텀 평균 허용(옵션)
- 라운딩: 결과 소수 둘째자리(0.01) 반올림 권장   
대답: 반올림 하지않고 소숫점 둘째자리까지 사용.

- 캡/바닥: 시공점수 상한 13점 검토, 하한(최소점) 여부 확인
대답: 시공점수 상한 13점이 맞아 하한점은 없어 그냥 계산식의 점수를 그대로 사용해.

- 방법 선택: 공고에 따라 경영점수 방법1 vs 방법2 선택 가능하도록 구성
대답: 공고에 따라 경영점수 방법을 이용해야해 공고마다 기초금액이 다르니 그만큼 기준치가 달라지는거라 시공점수 점수가 변경되거든
- 다음 작업: src/shared/formulas.defaults.json에 위 기본값 스키마로 반영 → 사용자 오버라이드 병합 로더 연결

## 질문
1) 영업기간 점수는 합산(가산)으로 이해했는데 맞나요? (곱셈 가중치 아님) 
  대답: 맞아 합산이야 부채7점 + 유동7점 + 영업기간1점 해서 총 15점 만점.

2) 부채/유동 비율 경계값은 “자체 ÷ 업종평균” 비율 기준이 맞나요?
  대답: 맞아 업종 자체 비율 / 업종평균  -> 즉 예를들어 전기공사 공고의 부채비율이라면 업종자체부채비율 /  전기 업종별평균부채비율 이 되는거지
3) 신용등급 유효기간/인정기관 범위를 알려주세요(공고 기준 반영 필요)
  대답: 인정기관은 없고 우리 엑셀에서 데이터 조회할때 등급이랑 날짜 유효기간 가져오잖아 사용자가 공고일을 입력을 하면  첫번째로 그 공고일 기준으로 신용평가 등급이 유효한지 판단하고 유효하다면 등급에따른 점수를 계산하는거야.

4) 시공점수 상한(13점) 적용 여부와 하한(최소점) 존재 여부 확인 부탁드립니다
  대답: 시공점수 상한 13점이 맞고 하한은 없어 그냥 계산한 점수 그대로야.
5) 기초금액은 추정가격/예정가격/기초금액 중 무엇을 사용하나요?
  대답: 기초금액을 사용해 이것도 사용자가 입력을 하는 방식으로 구현할거야. 그럼 그걸 기준으로 잡으면 돼
6) LH의 업종평균 매핑은 건설로 보면 될까요? 발주처별 평균 커스텀 필요성은?
  대답: 우린 건설이아니고 전기,통신,소방 공사야 그리고 발주처별 커스텀은  설정에서 적격검사 산출 식을 수정할수있도록 구현을 할거라면 커스텀이 들어간다고 봐야되는거야?
7) 반올림 규칙(소수 몇째, round/floor/ceil) 명확화 필요
  대답: 반올림은 하지말고 소숫점 둘쨰자리까지만 사용.






--- LH 요청목록 (리미트 걸려서 2일제한걸림.)



발주처는 고르지않는부분이 좌측메뉴바에 발주처별로 메뉴를 뺄거야
LH, 행안부, 조달청, 한국도로공사 등등 이렇게 발주처 별로 메뉴를 나누고
LH 메뉴 안에서 금액범위를 선택하는거로 생각중이야
LH를고르면 그 안에서 금액범위 50억미만, 50억~100억 이렇게 메뉴를 고르는거지. 

## 입력 항목
- 공종: 전기/통신/소방
- 공고번호, 공고명
- 기초금액, 추정가격 (숫자)
- 공고일 (달력에서 선택)
-  시공비율시 추정금액 기준 (숫자)
- 입찰참가자격금액 (이건  사용자가 입력한 추정가격을 그대로 가져다 쓰면돼)

### 제약/규칙
-  지역업체 의무:  어디지역, , 최소지역업체 의무비율 (%)
- 팀원수 :  최대 몇 개사 (1~5)
- 대표사고정, 지분제한,  제외/고정 업체리스트는 설정메뉴에서 셋팅값에 추가하는걸로 할게

### 후보 풀
- 지역(의무지역이 경기, 강원 이렇게 되어있어서 구체적인 지역은 상관없어 위에서 사용자가 정한 의무지역 안에만 들어오면돼
- 품질평가 기준: 품질평가점수가 80점 미만이면 0점이고 내가 json에 적은 품질평가 기준을 따르면되는거야
- 신용등급/재무 지표 필터는 업체를 조회할때 둘 중에 한가지만이라도 만점이라면 상관없어 그러니까 신용등급으로도 계산하고 재무지표로도 계산해야하는거지
- 후보상한은 좀 더 고려해보고 추가할계
