# 협정보드 최근 작업 요약 (2025-11-05)

## 구현 및 수정 사항
- **참가자격 옵션 확장**: 설정 화면과 협정보드에서 `참가자격 없음` 모드를 추가했습니다. 없음을 선택하면 금액 입력이 숨겨지고, 계산과 뱃지가 비활성화됩니다.
- **지역사/협정 슬롯 검색 개선**: 협정보드 슬롯 내 `업체 검색` 버튼으로 회사를 고르면 즉시 해당 슬롯에 배치되도록 정합 검증(사업자번호·기업명 fallback)을 보강했습니다.
- **담당자 배지 노출**: 협정보드 카드와 후보 목록에 추출된 담당자명을 뱃지로 표시해 상대 기업 정보 확인을 빠르게 할 수 있습니다.
- **복사 편의 기능 추가**: 협정 그룹별 경영점수, 5년 실적, 시평액을 탭 구분 형태로 클립보드에 복사하는 버튼을 제공했습니다.
- **행안부 30억~50억 실적 테이블**: 등급-퍼센트 기준(15점 만점)을 설정 기본값과 계산 로직에 반영했습니다.
- **참가자격 입력 UX**: 협정보드 및 설정 화면의 참가자격 입력을 양방향 동기화하고, 없음 모드일 땐 placeholder만 보이도록 다듬었습니다.

## 확인된 잔여 과제
1. **entryMode 전환 시 필드 잠금**: 없음 모드에서도 협정보드 헤더의 `참가자격` 금액을 입력하려고 하면 막히지만, 버튼 자체는 활성 상태입니다. 완전 비활성화가 필요한지 검토가 필요합니다.
2. **지역사 찾기 API 파라미터**: `entryMode = none`일 때도 조회 요청에 빈 참가자격 금액이 전달됩니다(서버 허용 확인 또는 클라이언트 단에서 제거 필요).
3. **협정 슬롯 검색 후 배치**: 대표사 외 지역사 케이스까지 수동 검증 권장. 새 fallback 매칭이 모든 데이터에 안전한지 QA 필요합니다.
4. **복사 기능 테스트**: 엑셀 붙여넣기 등 실환경에서 탭 구분 값이 예상대로 표현되는지 케이스별로 점검해야 합니다.

## 다음 세션 제안
- 없음 모드 도입에 따라 문서/교육 자료도 업데이트 필요 여부를 확인합니다.
- 후보 검색→협정 슬롯 배치 흐름을 실제 데이터로 재현해 오류 로그를 수집해주세요.
- 실적 점수 계산(행안부 30억~50억) 리그레션 테스트를 해 보고, 점수표가 다른 구간에 영향 없는지 확인 바랍니다.
