# 메일 작업자 거르기

## 목적
- 엑셀의 H열(작업) 정보로 작업자별 업체만 남기는 기능 설명
- 병합 셀로 인해 H열이 비어 보이는 행 처리 규칙 정리

## 사용 흐름
1) SMTP 프로필을 선택한 뒤 **불러오기**
2) 엑셀 파일 로드
3) 업체 목록의 **작업할업체만남기기** 버튼 클릭

> 프로필을 불러오기 전에 버튼을 누르면 "먼저 SMTP 프로필을 불러오세요." 안내가 표시됨

## 필터 기준
- SMTP 프로필 이름 기준으로 H열 값을 비교
- 비교 토큰:
  - 프로필 이름 전체 (공백 제거)
  - 프로필 이름의 첫 글자(성)
- H열 값에 위 토큰이 포함되면 해당 업체를 유지

예시:
- 프로필 이름이 "김희준"이면 `"김희준"` 또는 `"김"`이 포함된 작업자 행을 유지
- 프로필 이름이 "이"처럼 한 글자면 `"이"`만 비교

## 병합 셀 처리 규칙
엑셀 병합 셀은 첫 행만 값이 있고, 나머지 행은 빈 셀로 읽힘.

이 기능은 다음 규칙으로 처리함:
- H열에 값이 있는 행을 만나면 그 값을 기준 작업자로 갱신
- H열이 빈 행이면 **직전 작업자 값**을 그대로 사용

예시:
- H12에 `이`, H12~H16 병합
  - H12: 이
  - H13~H16: 빈칸이지만 **이**로 인식
- H17에 `박`, H17~H21 병합
  - H17: 박
  - H18~H21: 빈칸이지만 **박**으로 인식

## 주의사항
- H열이 비어 있으면 자동으로 위 작업자를 이어받기 때문에,
  의도적으로 비워 두려면 `-` 같은 구분 값을 넣는 것이 안전함
