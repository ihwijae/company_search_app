1. **협정보드 개선 방향**
   - 현재 협정 자동화보다 협정보드를 다듬는 것이 현장 대응에 더 현실적이라는 판단.

2. **한눈에 보기 UI 고민**
   - 목표: 협정보드에서 가로 스크롤 없이(또는 최소한의 스크롤로) 엑셀처럼 "대표/구성/지분/점수"를 한 화면에 보여주기.
   - 후보 아이디어:
     - 열 폭/숫자 포맷 축소(예: `123,456,789` → `12.3억`), 긴 텍스트는 말줄임+툴팁.

     - 행 클릭 시 아래쪽 패널이나 모달에서 상세 지표(점수/지역/세부 지분) 전체를 표시하는 방식.
   - 다음 세션에서 우선 폭 축소 및 그룹 토글을 적용해 본 뒤, 필요하면 전체 화면 모드나 확장패널을 검토할 것.
