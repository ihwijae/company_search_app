# 협정보드 작업 정리 (2026-01-08)

## 품질점수(LH 전용)
- LH 선택 시 품질점수 행을 추가(지분합계까지만 한 줄 추가, 신인도 이후는 rowSpan으로 병합).
- 품질점수 기본값: 100억 이상 88점, 그 외 85점(금액구간 선택 기준).
- 품질점수는 업체별 점수를 지분(%) 가중 평균으로 합산하여 `품질총점` 표시.
- `지분합계` 헤더에 구분선 추가 후 `품질총점` 라벨 표시.
- 품질 안내 문구(품질 기준표)를 품질 행 좌측 병합 셀에 표시:
  - LH 50억 미만: `품질 88점이상:3점/85점이상:2점/83점이상:1.5점/80점이상:1점`
  - LH 50억~100억: `90점이상:5점/88점이상:3점/85점이상:2점/83점이상:1.5점/80점이상:1점`
- LH에서 업체명 카드의 품질 배지/품질 텍스트는 제거.
- 대표사/구성사 배지 제거(모든 발주처 공통).

## 협정 테이블 표시 관련
- 업체명 길이는 줄바꿈으로 전체 표시(ellipsis 제거).
- 의무지역 업체명 셀 배경 노란색 표시.
- 예상점수 만점 여부는 연번 셀 배경(초록/빨강)으로 표시.
- 경영/실적 총점은 현재 점수만 표시하고, 만점이면 초록/아니면 빨강 볼드 처리.
- 품질 행 우측 영역 병합은 rowSpan 방식으로 해결(엑셀처럼 보이도록).

## 신인도/가능지분
- 가능지분 표시 조건: LH는 `시공비율기준금액`, 그 외는 `투찰금액` 입력 시만 표시.
- 가능지분 표시는 업체명 카드에만 표시(지분칸 표시 제거).

## 경영점수 가점(체크박스)
- 경영총점 옆에 `가점` 체크박스 추가(행 단위).
- 체크 시 경영점수에 +10% 가산, 상한 15점(발주처 max)으로 clamp.
- 총점 계산에도 반영.

## 순공사원가 가점(LH 전용)
- 헤더에 `순공사원가`, `A값` 입력칸 추가(LH만, 기초금액 아래).
- 헤더(투찰율 아래)에 `순공사원가가점` 표시 박스 추가.
- 테이블 하단 입찰점수 옆 `순공사원가가점` 컬럼 추가.
- R 계산 시 소수 4자리 반올림 적용.
- 최종 가점은 소수 2자리 절사.
- R이 0.88 초과 시 `올라탈수록 점수 깎임` 문구 표시.
- LH 50억 미만: PriceScore=70-4*ABS((0.88-R)*100), 기준 65점 초과분 가점.
- LH 50억~100억: PriceScore=50-2*ABS((0.88-R)*100), 기준 45점 초과분 가점.
- 상세 계산 정리: `docs/순공사원가.md` 참고.

## 실적/경영 기준 구간 결정 방식 변경
- 실적/경영 기준 구간은 **사용자 선택한 발주처+금액구간**만 사용.
- 추정가격/기초금액 입력과 무관하게 구간 라벨로 티어 선택.

## 설정/값 변경
- LH 실적점수: 50억 미만, 50억~100억 모두 13점 만점으로 수정(공식 반영).
- MOIS(행안부) 30억 미만/30~50 구간은 15점 만점으로 표시되도록 구간 선택 기준으로 변경.
