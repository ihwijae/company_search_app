- 공고번호/공고명: `K1`
- 개찰일: `M2`
- 의무지역/의무지분/참여업체수: `Q2`

## 업체명 표시 규칙
- 업체명 셀에 `업체명 + 가능지분(숫자) + 담당자` 순서로 표시.
- 예시: 가능지분 89%인 경우 `A업체 89\n담당자`

## 행 구조

- 2번째 협정: 기본 행 `7`, 품질점수 행 `8`
- 이후 동일 패턴 반복
