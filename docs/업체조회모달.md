# 업체조회 모달 UI 변경 안내

## 개요
- `CompanySearchModal`의 UI를 기존 내장 모달 스타일에서 전용 커스텀 모달 UI로 변경했습니다.
- 기능 로직(검색/선택/IPC 호출 등)은 그대로 유지하고, 디자인만 개선했습니다.

## 적용 파일 경로
- 컴포넌트: `src/components/CompanySearchModal.jsx`
- 스타일: `src/styles.css`

## 변경 내용 요약
- 전용 백드롭 및 모달 컨테이너(`company-search-backdrop`, `company-search-modal`)로 구조 분리
- 헤더(제목/설명/공종 배지) 레이아웃 추가
- 검색바 UI 개선(입력/버튼 높이, 포커스 스타일)
- 테이블 영역 카드형 스타일 적용
- 모달 애니메이션/그림자 적용

## 참고
- 동일 컴포넌트를 사용하는 모든 기능(협정보드, 설정 등)에도 동일한 UI가 적용됩니다.
- 블러(backdrop-filter)는 제거되어 불필요한 화면 흐림 없이 사용 가능합니다.
