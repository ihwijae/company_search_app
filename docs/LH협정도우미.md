엑셀협정도우미 - 엑셀에 채우기 기능에서 LH 발주처 쪽을 좀 수정해야 할 부분이있어

1. 지금 LH 경영상태 만점은 15점인데 17점으로 나오거든? 내 예상인데 경영상태 점수 구하는 기준에 "품질점수 2점"이 들어있는거같아 그래서 17점으로 나오는거같은데
이 틀린거야. 품질점수랑 경영점수는 완전히 다른 점수야. 내 생각이 맞다면 그렇게 수정해주고 아니라면 어떤 문제인지 알려줘


2. 기존 엑셀에 채우기 기능을 보면  업체명을 기준으로 +6이 지분, +13이 경영점수, +20이 실적액 이잖아?
하지만 LH는 여기에 품질점수라는게 추가로 있어 품질점수의 위치는 업체명셀을 기준으로 (r+1, c+6) 위치야 예를들어 업체명이 C5라면 품질점수는 I6인거야
업체를 조회하고 가져올때 이제 업체의 모든 데이터를 가져오잖아? 이때 만약 품질평가 쪽의 데이터가 있으면 그 데이터를 품질평가 점수에 넣어주면되는거야
만약 품질평가 란에 데이터가 없거나(공백) 없음이거나 이런 경우라면 그냥 기본점수인 85점을 넣어주면 돼
그리고 주의사항이  94.48 (25.05.01) 이런식으로 되어있다고 칠게 그럼 품질평가 점수가 94.48점 인건데 뒤에 날짜가 (25.05.01) 이지? 이러면 유효한 품질평가점수야
매년 5월1일마다 갱신이되거든. 그러니까 26년 5월1일 전까지는 유효한 점수인거지. 이걸 토대로 만약 (24.05.01)이런 내용이 있거나 하면 유효기간이 지난거니까 품질평가 점수가 있어도 기본점수인 85점이 들어가야하는거야 이해했어?
