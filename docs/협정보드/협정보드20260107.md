# 협정보드 작업 정리 (2026-01-07)

## 헤더 UI 재구성
- 발주처/금액구간/공종, 추정가격·기초금액, 사정율·투찰율, 공고일·투찰/시공비율금액을 2열 스택으로 재배치
- 공고번호/공고명 + 개찰일 묶음 구성, 참가자격금액과 산출방식 토글을 한 박스에서 처리
- 의무지역/의무지분 요약 카드 디자인 통일, 지역 선택은 별도 팝업 모달로 변경
- 헤더 전체 배경을 흰색으로 맞추고 여백을 최소화했으며 alert 메시지를 위한 배너 추가

## 협정 테이블 개선
- 업체 열을 100px까지 줄이고 지분/지분합계/실적/점수 계열 칸을 전체적으로 축소해 한 화면에 더 많은 열 표시
- 각 슬롯에 신인도 입력 칸 추가(숫자 입력 + 지분 반영 점수 표시), LH/PPS일 때만 노출
- 신인도 합 요약 열을 신인도 입력 옆으로 이동하고 열 폭을 95px로 고정해 겹침 해결
- 열 순서: 업체명 → 지분 → 지분합계 → 신인도 입력들 → 신인도 합 → 경영상태 → 경영 → 시공실적 → 실적(15점) → 입찰/예상점수

## 기타 UX
- 공종을 선택하지 않고 업체 검색을 시도하면 헤더 상단에 경고 배너를 띄워 공종 선택을 유도
- 의무지역 선택은 중앙 팝업에서 검색/체크/초기화/선택완료로 이뤄지며, 기존 인라인 패널 제거
