# 협정보드 인계 문서 (2026-01-12)

## 개요
행안부 30~50억 구간 기능/내보내기, 개찰일 입력 UI 개선, 협정보드 세부 로직 보완을 진행했습니다.

## 주요 변경사항
### 1) 행안부 30억~50억 협정보드 로직
- 실적 기준: 행안부 30억 미만과 동일(80% 미만 15점, 구간별 동일).
- 경영점수 표시: 업체별 경영점수는 15점 만점으로 표시.
- 경영총점: 최종 합계에서 10/15 스케일 적용(10점 만점).
- 영업기간 점수: 3년 이상 1점, 3년 미만 0.9점.
- 하도급 점수: 입찰점수 좌측에 하도급 칸 추가, 기본 5점.
- 가능지분 계산 기준: 시평액/투찰금액(조달청 50억 미만과 동일).
- 기본값 자동 계산: 기초금액 = 추정가격 × 1.1, 사정율 101.8, 투찰율 88.745, 투찰금액 자동.

### 2) 개찰일 입력 UI 개선
- 기존 `datetime-local` 대신 달력 + 오전/오후 + 시/분 직접 입력 UI로 변경.
- 분 선택 리스트 대신 직접 입력 방식으로 사용성 개선.

### 3) 행안부 30억~50억 엑셀 내보내기
- 템플릿 추가: `템플릿/행안부_30억~50억.xlsx`
- 템플릿 시트명: `양식`
- 시평액 컬럼: AP/AQ 시작(행안부 30억 미만과 차이점)
- 업체명에 가능지분 표시(100% 미만만 표시)
- 기초금액 헤더 셀: D1
- 기타 위치는 행안부 30억 미만과 동일
