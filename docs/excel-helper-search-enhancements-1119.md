# 엑셀 협정 도우미 검색 개선 정리

## 1. 중복 업체 선택 모달
- 동일한 업체명이 여러 건 검색되면 모달로 사용자에게 선택권 제공.
- 검색 결과마다 사업자번호, 대표자, 지역, 담당자 뱃지까지 표시.
- 모달에서 고른 업체는 `companyConflictSelections` 상태로 보관하여 향후 동일 이름에도 자동 적용.

## 2. 이름 정규화 확장
- `normalizeName` 단계에서 `이앤 → 이엔`, `앤 → 엔` 치환을 추가.
- 줄바꿈을 포함한 이름(`\r\n`)도 숫자/담당자를 정확히 제거하도록 정규식 보완.
- 덕분에 `연일전력81.59\r\n윤명숙` 같은 셀에서도 순수 업체명만 추출.

## 3. 오타 자동 재검색
- 1차 대량 검색에서 매칭되지 않은 업체명을 따로 모음.
- 각 이름에 대해 `이앤↔이엔` 변환 후보를 생성해 2차 `searchManyCompanies` 호출.
- 2차 검색 결과도 기존 매핑(Map)에 병합하여 이후 로직(모달 포함) 그대로 활용.
- 2차에서도 찾지 못한 업체는 기존과 동일하게 원본 셀 값을 그대로 사용.
