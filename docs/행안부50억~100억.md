# 행안부 50억~100억 기준 및 엑셀 매핑

## 템플릿
- 파일: `템플릿/행안부_50억~100억_템플릿.xlsx`
- 시트명: `양식`
- 시작 행: `5`
- 최대 행: `68`
- 템플릿 키: `mois-50to100`

## 협정보드 헤더 기본 동작
- 추정가격 입력 시 기초금액 자동 계산: `기초금액 = 추정가격 * 1.1`
- 기본 투찰율: `87.495`
- 기본 사정율: `101.6`
- 실적만점금액: `추정가격 * 1.7`
- 투찰금액 자동 계산(행안부 50~100):
  - `예정가격 = 기초금액 * 사정율`
  - `투찰금액 = ((예정가격 - A값) * 투찰율) + A값`
- 헤더 입력칸 추가:
  - `순공사원가`
  - `A값`
- 안내문구:
  - 투찰금액 아래 `A값 반드시 확인하세요!` (빨간색/볼드)

## 점수 기준

### 1) 실적점수 (15점 만점)
- `실적점수 = 실적계수 * 15`
- `실적계수 = 최근 3년간 실적 / (추정가격 * 1.7)`
- 최근 3년간 실적은 협정 참여업체 지분 반영 합산
  - 예: `(A사 3년실적 * 지분) + (B사 3년실적 * 지분) + ...`
- 주의: 행안부 50~100은 **5년 실적이 아니라 3년 실적** 사용

### 2) 경영상태 점수 (15점 만점 환산)
- 항목 합산 후 `* (15/21)` 환산
- 부채비율 원점수
  - 50% 미만: 10.0
  - 50% 이상 75% 미만: 9.3
  - 75% 이상 100% 미만: 8.6
  - 100% 이상 125% 미만: 7.9
  - 125% 이상: 7.2
- 유동비율 원점수
  - 150% 이상: 10.0
  - 120% 이상 150% 미만: 9.3
  - 100% 이상 120% 미만: 8.6
  - 70% 이상 100% 미만: 7.9
  - 70% 미만: 7.2
- 영업기간 원점수
  - 5년 이상: 1.0
  - 3년 이상 5년 미만: 0.9
  - 3년 미만: 0.8

### 3) 신용평가 점수 (15점 만점 환산)
- 신용평가 원점수 후 `* (15/21)` 환산
- 원점수 구간
  - AAA ~ BB0: 21.0
  - BB-: 20.0
  - B+, B0, B-: 19.0
  - CCC+, CCC0, CCC-, CC, C, D: 16.0

## 고정점수(행안부 50~100)
- 입찰점수: `45`
- 하도급: `10`
- 자재: `10`
- 협정보드 표시 순서: `하도급 -> 자재 -> 입찰점수`

## 실적 표시 규칙
- 협정보드 시공실적 라벨: `3년 실적`
- 후보/협정 계산/복사/표시 모두 50~100 구간에서는 3년 실적 우선 사용

## 엑셀 헤더 셀 매핑
- 추정가격: `D1`
- 기초금액: `D2`
- 투찰금액: `F2`
- 공고번호/업체명: `N1`
- 개찰일: `O2`
- 의무지역/의무지분/참여개사: `U2`
- 참가자격 안내문구: `AA2`
  - 형식: `※입찰참가자격 : 시공능력평가액 {금액}원 이상 ({비율반영|단순합산})`
  - `entryMode`가 `none`이거나 참가자격금액이 없으면 공란
- 참가자격금액(숫자): `AQ3`

## 엑셀 슬롯 컬럼 매핑
- 업체명: `C, D, E, F, G`
- 지분: `I, J, K, L, M`
- 경영상태(점수): `P, Q, R, S, T`
- 시공실적: `W, X, Y, Z, AA`
- 시평액: `AO, AP, AQ, AR, AS`
  - 협정1 대표사: `AO5`
  - 협정1 구성사1: `AP5`
  - 협정2 대표사: `AO6`
- 승인 컬럼: `B`
