# 2025-02-14 작업 정리

## 협정보드 / 지역사 찾기 창 개선
- 협정보드 독립 창 초기 크기를 크게 조정하여(최대 1480×1040) 기본 화면에서 더 넓은 작업 영역 확보.
- 지역사 찾기 팝업 기본 크기를 확대(최대 1720×1080)하고 중앙에 배치해 반복적인 창 크기 조절 없이 작업 가능하도록 개선.

## 협정보드 화면 UX 정비
- 협정보드 상단 미리보기 영역(엑셀 내보내기, 협정 문자 생성 등) 버튼 줄을 `sticky`로 고정해 스크롤 시에도 상시 노출.
- 루트 컨테이너의 wheel 이벤트를 가로채 메인 협정 목록이 부드럽게 스크롤되도록 처리(사이드 리스트 위에서는 기존 동작 유지).

## 엑셀 복사 포맷 / 가능지분
- 엑셀 붙여넣기용 복사 시 업체 셀에 `업체명 ↵ 가능지분 ↵ 담당자` 형식으로 줄바꿈 구성.
- 지분(%) 열은 사용자가 입력한 숫자를 그대로 유지.
- 가능지분 계산을 발주처별 금액 기준(조달청=투찰금액, LH=시공비율 기준금액 등)으로 다시 정리하고, 숫자 파싱 시 콤마 제거 처리.

## 계약 설정 페이지 레이아웃
- 자동 구성 제안·확정/내보내기 패널을 제거하고 후보 현황 패널 및 하단 액션 푸터(`지역사 찾기`, `협정보드 열기`)로 재구성.
- 후보가 없을 때 안내 문구, 후보가 있을 때 요약 정보가 나타나도록 구성.

## 기타
- 새로 도입한 `useCallback` 등 훅 import 고정.
- 관련 스타일(`candidate-summary`, `action-footer` 등) 추가.
