# 조달청 50억 미만 엑셀 매핑 정보

## 템플릿
- 파일: `템플릿/조달청50억미만.xlsx`
- 시트: `양식`

## 헤더 매핑
- 추정가격: `D1`
- 기초금액: `D2`
- 투찰금액: `F2`
- 공고번호/공고명: `M1`
- 개찰일: `P2`
- 의무지역: `W2`

## 업체명 표시 규칙
- 조달청 50억 미만 내보내기 시 업체명 셀에 가능지분을 숫자만 붙여 표기.
- 예시: 가능지분 89%인 경우 `A업체 89`
- 담당자가 있으면 다음 줄에 표기: `A업체 89\n담당자`

## 시평액(대표사/구성원)
- 대표사: `AO`
- 구성원1: `AP`
- 구성원2: `AQ`
- 구성원3: `AR`
- 구성원4: `AS`
- 행안부 30억 미만도 동일한 위치(AO~AS)에 매핑됨.
