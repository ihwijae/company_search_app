# 엑셀 도우미 기능 구현 현황

## 완료된 사항
- 엑셀 도우미를 메인 창과 분리된 독립 창으로 동작시키고, PowerShell 연동 시 한글 경로/시트명도 안전하게 전달하도록 Base64 인코딩을 적용했습니다.
- 엑셀 기준 셀 동기화 → 업체 검색/선택 → 지분 입력 → 엑셀에 채우기 → 협정 문자 생성까지 한 화면에서 처리할 수 있는 단일 플로우를 구축했습니다.
- 엑셀 오프셋은 발주처별 고정 규칙(+0, +6, +13, +20, +38 / LH 추가 오프셋)을 그대로 사용하며, 지분 값은 입력된 % 값을 자동으로 0.x 형태로 변환해 엑셀 수식과 맞물리도록 했습니다.
- 협정 문자 생성은 현재 선택된 기준 셀과 그 오른쪽 최대 5개 슬롯을 직접 읽어서 대표사/구성사를 구성한 뒤, 기존 협정 문자 모듈을 활용해 클립보드에 복사합니다.

## 다음에 진행할 작업
1. **업체명 복사 규칙 개선**: `(주)` 제거, 담당자/지분 병기 등 요구 사항에 맞춰 복사 문자열을 재구성해야 합니다.
2. **조건부 서식/색상 지원**: 사용자가 정의한 규칙(예: 경영점수 미달 시 셀 배경 색상 변경)을 적용해 붙여넣을 수 있도록 셀 서식 조정 로직을 추가해야 합니다.
3. **관리점수 계산 고도화**: 현재는 원본 데이터의 경영점수를 그대로 사용합니다. 필요 시 지역사찾기/협정보드에서 쓰는 부채·유동·영업기간/신용기반 계산 로직을 그대로 이식해, 입력 항목을 추가하거나 자동 채우는 방식을 검토해야 합니다.
4. **추가 데이터 입력**: 향후 요청될 업체명/담당자/지분 등 복사 포맷 변경이나, 엑셀 내 검증 규칙(예: 합계 100% 검증)도 문서화해 단계별로 구현합니다.

> 참고: 1~2번은 사용자 세션에서 추가 규칙을 전달해준 뒤 구현 예정이며, 이 문서는 그 인수인계용 초안입니다.
