# 카카오톡 전송 변경 요약 (2026-01-26)

## 배경
- 카카오톡 자동전송(Helper.exe 기반) 구현 과정에서 실행/런타임 오류가 반복 발생.
- 자동전송을 제거하고, 사용자가 카카오톡에서 직접 보내는 방식(수동 전송)으로 전환.
- 대신 “담당자별로 보낼 문자 정리/자동 매칭/편집”은 유지하여 편의성 제공.

## 현재 결정 사항
- **카카오톡 자동전송 기능 제거**
- **문자 분리 / 담당자 자동매칭 / 메시지 편집 UI는 유지**
- 전송은 **사용자가 카카오톡에서 수동으로 진행**

## 적용된 변경 사항 (코드)
- 자동전송 백엔드 제거
  - `main.js`에서 KakaoAutomationService 및 IPC(`kakao:send-batch`) 제거
  - `preload.js`에서 `electronAPI.kakao.sendBatch` 제거
- 카카오 자동화 헬퍼 삭제
  - `src/main/features/kakao/` 하위 `.cs`, `.csproj`, `kakaoAutomation.js`, 빌드 README 삭제
- 카카오 전송 화면 유지
  - `src/view/features/kakao/pages/KakaoSendPage.jsx`
  - 자동전송 버튼 제거
  - 안내 문구 추가 및 볼드/폰트 크기 상향

## UI 변경 사항
- 카카오 전송 화면 하단에 다음 문구 표시(강조):
  - “자동 전송 기능은 제거되었습니다. 카카오톡에서 수동으로 전송해 주세요.”

## 다음 세션 작업 예정 (반자동화)
- 자동전송 없이도 담당자별로 **전송 준비가 끝난 텍스트를 정리/복사**할 수 있게 개선
- 요구사항 결정 후 구현 예정:
  - 출력 형식(텍스트/CSV/엑셀)
  - 담당자 기준/채팅방 기준 묶음 방식
  - 카카오톡 붙여넣기 친화 포맷
