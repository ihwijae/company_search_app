# 협정 금지 규칙

## 적용 대상
- 모든 발주처 공통
- 전기/통신/소방 공통
- 설정 화면(협정 규칙 편집)과 무관하게 **고정 config**만 참조

## 규칙 적용 위치
- 협정보드에서 같은 협정(같은 행/그룹) 내 조합을 검사
- 결과 메시지는 **업체명 컬럼 맨 끝의 비고 칸**에 표시

## 설정 파일 경로
- `src/shared/agreements/banConfig.js`

## 규칙 요약
### 1) 동일 담당자 중복 금지
- 동일 담당자가 속한 업체가 2개 이상이면 금지
- 담당자명은 후보 데이터의 담당자 필드/비고에서 추출

### 2) 담당자 조합 금지
- 아래 담당자 2명 조합이 같은 협정에 있으면 금지
- 네 명(박성균/나의상/송종윤/안영식)은 **서로 모든 2명 조합 금지**

담당자 조합 목록:
- 윤명숙 + 김대열
- 윤명숙 + 조정
- 김장섭 + 송종윤
- 김장섭 + 조재진
- 윤한봉 + 이재웅
- 구본진 + 송종윤
- 구본진 + 나의상
- 조재진 + 안태혁
- 박성균 + 나의상
- 박성균 + 송종윤
- 박성균 + 안영식
- 나의상 + 송종윤
- 나의상 + 안영식
- 송종윤 + 안영식

### 3) 업체/담당자 조합 금지
- 아래 조합이 같은 협정에 있으면 금지
- 업체명은 DB에 등록된 업체명을 그대로 사용

조합 목록:
- 코레일테크㈜ + 보명산업개발㈜
- 코레일테크㈜ + 해성테크㈜
- 에코엠이엔씨㈜ + 에스지씨이앤씨㈜
- 태평양이앤씨(주) + 에스지씨이앤씨㈜
- ㈜성전사 + 에스지씨이앤씨㈜
- ㈜김앤드이 + 나의상(담당자)
- 윤명숙(담당자) + 도원이엔아이
- 김장섭(담당자) + ㈜왕선전설
- 김장섭(담당자) + ㈜기전사
- 김장섭(담당자) + ㈜보원엔지니어링
- ㈜벽산전기 + 세명건설㈜
- ㈜세영이앤씨 + 성원전기
- ㈜트래콘건설 + 이화공영㈜
- 구본진(담당자) + 대명지이씨㈜
- 구본진(담당자) + ㈜도화엔지니어링
- ㈜남양계전 + ㈜이엘케이
- ㈜부현전기 + ㈜하나전기
- ㈜부현전기 + ㈜유리시스템즈
- ㈜대광건영 + 김화성(담당자)
- ㈜대한종합산전 + ㈜코원건설

## 유지보수
- 규칙 변경은 `src/shared/agreements/banConfig.js` 수정
- 협정보드 UI는 해당 config만 참조
