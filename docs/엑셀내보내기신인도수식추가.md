# 엑셀 내보내기 신인도 수식 추가

## 목적
협정보드에서 입력한 신인도(가점) 값을 엑셀 내보내기 시 합산 값으로만 쓰지 않고,
지분 셀과 곱한 수식으로 기록하여 검토자가 산출 근거를 확인할 수 있도록 한다.

## 동작 방식
- 신인도 합계 셀에 각 업체의 `신인도점수 * 지분셀`을 더한 수식을 넣는다.
- 신인도 점수가 없는 업체는 수식에 포함하지 않는다.
- 지분 값은 템플릿의 지분 컬럼을 그대로 사용한다.
- 템플릿 설정에 `summaryColumns.credibility`가 지정된 경우에만 적용된다.

## 수식 예시 (LH 템플릿)
- 협정 3번 (행 9):
  - A업체 1.5점만 있는 경우: `=1.5*I9`
  - A업체 1.5점, B업체 0.5점: `=1.5*I9+0.5*J9`

## 국가철도공단 스케일 반영
국가철도공단은 신인도 합계에 스케일을 적용한다.
- 50억 미만: `신인도합계 * 0.5/3`
- 50억~100억: `신인도합계 * 0.9/3` (템플릿 추가 시 설정 필요)

예) 50억 미만, 협정 1행, A업체 3점:
- `= (3*I5) * 0.5/3`

## 템플릿 설정 반영
- 한국도로공사(EX) 50억 미만: `summaryColumns.credibility = 'AD'`
- 국가철도공단(KRAIL) 50억 미만: `summaryColumns.credibility = 'AK'`,
  `credibilityScaleExpr = '0.5/3'`

- 템플릿 설정: `main.js`
