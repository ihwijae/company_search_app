# 단독입찰 가능 여부 확인 (협정보드)

## 목적
협정보드에서 업체를 추가했을 때 **단독입찰 가능** 여부를 즉시 판단해 "단독가능" 표시를 제공한다.
업무 중 단독 가능 여부를 놓쳐 협정을 불필요하게 진행하는 상황을 예방한다.

## 적용 위치
- 화면: 협정보드 업체명 셀 태그 영역
- 표시: 조건 충족 시 `단독가능` 태그 표시
- 구현 파일:
  - 로직/표시: `src/view/features/agreements/components/AgreementBoardWindow.jsx`
  - 스타일: `src/styles.css` (`.excel-tag-single-bid`)

## 단독 가능 판정 규칙 (공통)
아래 조건을 **모두 충족**해야 단독입찰 가능으로 판단한다.

1) **참가자격금액 조건**
- 참가자격금액(헤더 입력)이 존재하는 경우
- 업체의 시평액 ≥ 참가자격금액이어야 함

2) **의무지역 조건**
- 사용자가 헤더에서 설정한 의무지역과 업체 지역이 일치해야 함
- 의무지역 미설정 시 지역 조건은 통과로 처리

3) **실적 만점 조건**
- 협정보드 헤더에 표시되는 "실적만점금액"을 기준으로 판단
- 업체의 5년 실적 금액 ≥ 실적만점금액이면 실적 만점으로 간주

4) **경영점수 만점 조건**
- 협정보드에서 계산된 경영점수가 **만점(관리 기준 최대치)** 이어야 함
- 협정보드 계산 로직과 동일한 기준으로 판단

> 참고: 단독 가능 여부 판단 시 참여지분은 **100%** 로 가정한다.

## 발주처별 규칙
모든 발주처가 위 공통 조건을 사용한다.
- 행안부 30억 미만: 실적 만점 + 경영 만점
- 행안부 30~50억: 실적 만점 + 경영 만점
- 조달청 50억 미만: 실적 만점 + 경영 만점
- LH 50억 미만: 실적 만점 + 경영 만점 (순공사가점/가점은 고려하지 않음)
- 한국도로공사 50억 미만: 실적 만점 + 경영 만점
- 국가철도공단 50억 미만: 실적 만점 + 경영 만점

## 실적만점금액 자동 산출 규칙
협정보드 헤더의 "실적만점금액"은 발주처/금액대별 규칙에 따라 자동 산출된다.

- 조달청 50억 미만: `기초금액 × 1배`
- 행안부 30억 미만/30~50억: `추정가격 × 80%`
- LH 50억 미만: `기초금액 × 1배`
- LH 50~100억: `기초금액 × 2배`
- 국가철도공단 50억 미만:
  - 전기/통신: `기초금액 × 1배`
  - 소방:
    - 추정가격 < 30억: `기초금액 × 2배`
    - 추정가격 ≥ 30억: `기초금액 × 3배`
- 한국도로공사 50억 미만: `기초금액 × 1배`

## 단독가능 태그 표시 조건 (협정보드 내부 계산)
- 참가자격금액 조건 OK
- 의무지역 조건 OK
- 실적 만점 조건 OK
- 경영점수 만점 조건 OK
→ 위 조건을 모두 만족하면 `단독가능` 태그 표시

## 구현 메모
- 판정 로직은 협정보드 내부에서 계산하며, 지역사찾기 결과의 `singleBidEligible`를 사용하지 않는다.
- 실적만점금액은 협정보드 헤더에 자동 계산된 값을 그대로 사용한다.
- 경영점수 만점 판단은 협정보드에서 사용하는 관리점수 계산 로직과 동일한 기준을 사용한다.
