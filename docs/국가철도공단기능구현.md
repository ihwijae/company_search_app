# 국가철도공단 기능 구현 현황

## 경영상태 계산 기준
- `src/shared/formulas.defaults.json`에 국가철도공단(`id: krail`) 전용 규칙을 추가했다.
  - 50억 미만 / 50~100억 두 구간을 정의해 부채·유동·영업기간 점수를 문서(`docs/국가철도공단경영상태기준.md`) 기준으로 구성했다.
  - 신용평가 테이블은 35점 만점을 15점으로 환산한 점수를 저장하고, 기존 로직과 동일하게 신용 유효기간 만료 시 점수 사용을 차단한다.
  - 실적(`performance`) 밴드는 조달청과 동일한 ratio-bands를 임시로 사용 중이다.

## 엑셀 협정 도우미 개선사항
- 발주처 선택 시 국가철도공단이면 기술자 입력 섹션이 열린다.
  - 각 기술자는 등급/경력계수/관리능력계수/인원수를 개별 행으로 입력하며, 점수는 `등급×경력×관리×인원수`로 합산한다.
  - 합산된 기술자 점수는 실시간으로 표시되고, 엑셀에 쓰는 값으로도 활용된다.
- 엑셀 동기화 오프셋
  - 지분(I), 경영상태(P), 실적(W)은 기존 발주처와 동일하게 배치한다.
  - 기술자 점수는 기준 셀(C열)에서 `+27` 열(AD)에 기록한다.
  - 시평액 열은 발주처/구간별로 다르게 설정했다.
    - 50억 미만: C 기준 `+45` 열(AV)
    - 50억~100억: C 기준 `+46` 열(AW)

## 남은 확인 사항
- 기술자 점수/시평액을 포함한 엑셀 쓰기 동작을 실제 템플릿에서 각각의 구간(50억 미만, 50~100억)에 대해 수동 검증해야 한다.
- formulas defaults 변경분이 evaluator와 문자 생성 플로우에 정상 반영되는지 QA 필요.
