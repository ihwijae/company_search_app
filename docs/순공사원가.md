# 순공사원가 가점 정리

## 공통 입력값
- Base: 기초금액
- NetCost: 순공사원가
- A: A값

## 예정가격(사정율 범위, 천원 미만 절상)
- 사정율 범위: 0.988 ~ 1.012
- 천원 단위 올림(엑셀 ROUNDUP, -3)
- ExpectedMin = ROUNDUP(Base * 0.988, -3)
- ExpectedMax = ROUNDUP(Base * 1.012, -3)

## 순공사원가 반영 입찰가격
- Bid = NetCost * (Expected / Base) * 0.98
- BidMin = NetCost * (ExpectedMin / Base) * 0.98
- BidMax = NetCost * (ExpectedMax / Base) * 0.98

## 비율 R (소수 4자리 반올림)
- R = (Bid - A) / (Expected - A)
- Rmin = ROUND((BidMin - A) / (ExpectedMin - A), 4)
- Rmax = ROUND((BidMax - A) / (ExpectedMax - A), 4)

## 입찰가격점수
- LH 50억 미만:
  - PriceScore(R) = 70 - 4 * ABS((0.9 - R) * 100)
- LH 50억~100억:
  - PriceScore(R) = 50 - 2 * ABS((0.9 - R) * 100)

## 가점 산출
- LH 50억 미만: 기준 65점 초과분만 가점
- LH 50억~100억: 기준 45점 초과분만 가점

- Bonus(R) = PriceScore(R) - 기준점
- FinalBonus = max(0, min(Bonus(Rmin), Bonus(Rmax)))
- 사정율 2개 경우 중 더 불리한(작은) 가점을 사용
- 최종 가점은 소수 2자리 **절사**

## 안내 문구 표시 조건
- Rmin 또는 Rmax가 0.9보다 크면
  - 헤더에 "올라탈수록 점수 깎임" 표시

## UI 표시 위치
- 헤더(투찰율 아래): 순공사원가가점 표시(0 또는 계산값)
- 테이블 하단: 입찰점수 옆 컬럼에 순공사원가가점 표시
