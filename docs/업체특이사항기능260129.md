# 업체특이사항 기능 정리 (2026-01-29)

## 개요
- 업체별 특이사항을 등록/조회/수정/삭제하는 전용 페이지를 추가했습니다.
- 공종/지역/업체명/사업자번호 필터링과 JSON 저장/내보내기/가져오기까지 지원합니다.

## 주요 기능
### 1) 데이터 관리
- 등록/수정/삭제 지원
- 자동 저장 (변경 시 약 300ms 후 저장)
- 내보내기/가져오기 (JSON)

### 2) 저장 구조
- 전용 파일: `company-notes.json` (userData 경로)
- 내보내기 파일 형식도 동일 구조
- 가져오기 시 기존 데이터 전체 덮어쓰기

### 3) 필터링
- 공종(전기/통신/소방, 전체)
- 지역
- 업체명/사업자번호
- 대표님업체만 보기
- 기타만 보기
- 업체명/사업자번호 검색 시 기타 여부와 무관하게 전체 노출

### 4) 대표님업체 기능
- 대표님업체 체크 → 행 강조 및 배지 표시
- 대표님업체 공통사항(헤더 버튼)
  - 전용 모달에서 공통 문구 관리
  - 공통 문구는 대표님업체 전체에 적용되며, 특이사항 칸에서 “공통” 배지 클릭 시 토글 표시

### 5) 기타 등록 기능
- 등록 모달에서 “기타 등록” 체크박스 제공
- 체크 시 지역 선택 비활성화
- 리스트에서 지역 칸에 파란색 “기타” 표시

### 6) 단독 상태
- 단독제외 / 단독이여도가능 / 없음
- 배지 표시

### 7) 여부묻기
- 물어보고 사용 / 안물어보고 사용가능 / 없음
- 배지 표시 (특이사항 칸에서만 표시)

### 8) 담당자 배지
- 업체 조회 시 담당자 자동 저장
- 리스트에 업체명 아래 담당자 배지 표시
- “담당자 배지 갱신” 버튼 추가
  - 전체 항목을 DB 기준으로 재조회
  - 변경된 항목만 갱신

### 9) UI/UX
- 전용 모달 UI(특이사항 등록, 대표님업체 공통사항)
- 업체 조회 모달 커스텀 UI 적용
- 다크톤 사이드바 색상 적용
- 줄바꿈 메모 표시
- 행 강조/배지 스타일 개선

## 추가 구현 사항
- 대표님업체 공통사항 모달
  - 공통 문구 추가/삭제 가능
  - 공통 문구는 반복 출력 대신 배지 토글 방식
- 필터 초기화 옆에 대표님업체 공통사항 버튼 배치

## 파일 경로
- 페이지: `src/view/features/notes/pages/CompanyNotesPage.jsx`
- 스타일: `src/styles.css`
- 라우팅/메뉴: `src/App.jsx`, `src/components/Sidebar.jsx`, `src/shared/navigation.js`
- IPC 저장/로드: `main.js`, `preload.js`
- 문서: `docs/업체조회모달.md`

## 유의사항
- 공통 문구는 대표님업체 전용 (라벨: “공통”)
- 기타 등록은 일반 공통 개념 (라벨: “기타”)
- 업데이트 설치 시 userData 경로 유지되면 데이터 보존됨
- 다른 PC/계정에서는 별도 공유 필요
